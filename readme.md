# Sentiment Analyzer

A web app that analyzes sentiment of any text dataset using NLP.

## What it does
- Upload any CSV file
- Select which column contains the text
- App analyzes every row automatically
- Shows Positive, Negative, Neutral breakdown
- Shows a pie chart of results
- Shows most positive and most negative text

## Technologies Used
- Python
- TextBlob
- Streamlit
- Plotly
- Pandas

## How to Run
pip install -r requirements.txt
python -m streamlit run app.py
